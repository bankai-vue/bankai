# docs (reserved)

This workspace folder is a **reserved stub** for the bankai-vue documentation
site, which — per [`SPEC.md` §4.15](../SPEC.md) — will be built **fully on Nuxt +
bankai-vue itself** (not VitePress), as the primary dogfooding and live SSR/SSG
proof.

It is intentionally empty for now. The docs site is scaffolded later in the
roadmap, once enough components exist for the docs shell (layout, nav, theme
toggle, code block, tabs) to be built _with_ bankai-vue.
