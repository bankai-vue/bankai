Add `BankaiButton` — the first bankai-vue component. It renders a native `<button>` with typed `variant`, `size`, and `type` props, exposes the `bankai-button` class plus `data-part`/`data-bankai-*` styling hooks, and ships no CSS (style it through the parts, or with the `@bankai-vue/theme-bankai` theme).
