Style `BankaiCodeBlock` — paints `.bankai-code-block` as a raised code panel: the house monospace stack, an `--bankai-text-size-sm` code size with a comfortable line-height, the `surface` background, and the house corner radius. The `<pre>` scrolls horizontally on its own (`overflow-x`) so a long line never widens the layout, the copy toolbar is pinned to the block's top inline-end corner (outside the scrolling `<pre>`, so it stays put while the code scrolls), and the `role="status"` live region is visually hidden (sr-only). Every rule is zero-specificity (`:where()`) so a consumer's own CSS or utility classes override it without `!important`, and every value rides a `--bankai-code-block-*` custom property (declared under `:where(:root)`) — `--bankai-code-block-bg`, `--bankai-code-block-font-family`, `--bankai-code-block-font-size`, `--bankai-code-block-line-height`, `--bankai-code-block-radius`, `--bankai-code-block-padding-block`, `--bankai-code-block-padding-inline`, `--bankai-code-block-copy-offset` — so the panel is retuned by overriding a single property.
